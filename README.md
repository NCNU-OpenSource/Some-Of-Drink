## 操作教學
----遊戲開始----

A方選擇要出的拳(剪刀:w, 石頭:d, 布:x)

B方選出要出的拳(剪刀:u, 石頭:k , 布:m)


----選出優勝者----

控制水管方向(v向右, b向左)

A獲勝控制馬達(a噴水, s停止)

B獲勝控制馬達(h噴水, j停止)
